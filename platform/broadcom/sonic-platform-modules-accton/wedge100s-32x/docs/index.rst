Wedge 100S-32X SONiC Platform
==============================

Platform documentation for the Accton Wedge 100S-32X running SONiC.
This platform uses the Broadcom Tomahawk (BCM56960) ASIC with 32 x 100G
QSFP28 ports and an OpenBMC AST2500 baseboard management controller.
